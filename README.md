# CareS - 健康提醒助手

一个轻量级的护眼/久坐/喝水提醒程序，带内存清理功能。

## 功能

- 👁️ **护眼提醒** - 定时提醒休息眼睛
- 🪑 **久坐提醒** - 定时提醒起来活动
- 💧 **喝水提醒** - 定时提醒喝水
- 🧹 **内存清理** - 一键清理系统内存
- 🔔 **声音提示** - 提醒时播放声音
- 📌 **托盘常驻** - 关闭窗口后最小化到托盘

## 使用方法

### 直接运行（需要Python环境）

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### 打包成exe

```bash
# 双击运行 build.bat
# 或者手动执行：
pip install -r requirements.txt
pyinstaller --onefile --windowed --name "CareS" main.py
```

打包完成后，`dist\CareS.exe` 就是独立的可执行文件。

## 功能说明

### 提醒设置
- 每个提醒可以独立开关
- 间隔时间支持快捷选择（10-60分钟）
- 支持自定义输入（1-60分钟）
- 弹窗提醒3秒自动消失

### 内存清理
- 点击主界面"清理内存"按钮
- 或右键托盘图标选择"清理内存"

### 托盘菜单
- 显示主窗口
- 清理内存
- 退出程序

## 系统要求

- Windows 7/8/10/11
- Python 3.8+（如果直接运行）

## 注意事项

- 关闭窗口会最小化到托盘，不会退出程序
- 要完全退出请右键托盘图标选择"退出"
- 内存清理需要管理员权限才能完全生效
